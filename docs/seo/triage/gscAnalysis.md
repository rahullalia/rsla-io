The March impression spike is partly from Batch 2 posts (12 Claude Code posts published Mar 2 to 12) getting indexed. If GA4 showed a traffic drop, it likely came from **Google Discover** (not web search) or was within noise given the extremely low click volume.

## Key Takeaways for Triage

1. **GHL is the cash keyword.** go-high-level-pricing and go-high-level-new-features-2025 account for 65,000+ impressions combined. These MUST be rewritten with deep GHL expertise.

2. **Claude Code is the growth bet.** Only 10 days of data but already showing position 8 to 10 for comparison queries. These posts need Rahul's daily-user experience injected.

3. **All vertical content (salon, HVAC, restaurant, RE) generates zero clicks.** Keep only the verticals where Rahul has actual client experience (salons via salon clients, restaurants via Spice on a Slice).

4. **Cold email posts have no authority signal.** 1,002 impressions but zero clicks and weak zone. Noindex unless external backlinks data shows otherwise.

5. **The "traffic drop" may be Discover, not web search.** Web impressions are UP. Rahul should check GA4 to see if the drop was in Discover, Direct, or Referral channels. If Discover, need separate GSC Discover export.

6. **CTR is the real problem, not impressions.** The site ranks on page 1 for dozens of GHL queries but converts at 0.02% CTR. Meta titles and descriptions need to be more compelling (experience-driven, specific numbers, opinionated).
