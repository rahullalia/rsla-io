## Product Clarity System

Anthropic has multiple products and people confuse them constantly. Every post must be crystal clear about which product it covers. This is a ranking advantage — most competitors are vague.

**The products:**
- **Claude.ai** — The web app at claude.ai. Chat interface, Projects, Artifacts. Free, Pro ($20/mo), Max tiers.
- **Claude Desktop App** — Native Mac/Windows app. Same as Claude.ai but local. Includes MCP support.
- **Claude Code** — CLI tool for developers. Terminal-based. Agentic coding. Reads/writes files, runs commands.
- **Claude API** — Developer API. Pay-per-token. For building apps on top of Claude.
- **Claude Co-Work** — Background agent inside Claude.ai that works on tasks asynchronously.

**Post 0 is the anchor.** Every other post links to it when product confusion might exist.

**Product tags on every post:**
- `[CLAUDE CODE]` — About Claude Code specifically
- `[CLAUDE.AI]` — About the web/desktop app
- `[ALL PRODUCTS]` — Covers multiple Claude products (pricing, capabilities, company)
- `[API]` — About the developer API
- `[COMPANY]` — About Anthropic the company

### Featured Image Generation (Gemini API)

Every blog post needs a custom featured image. Generate using the Gemini API:

**Process:**
1. After the blog post is drafted and approved, generate a featured image
2. Use `google/gemini-2.0-flash-exp` via the Gemini API (image generation model)
3. Prompt pattern: Create a clean, modern blog header image related to [topic]. Style: minimal, tech-forward, blue-gray palette (#0F172A, #0070F3, #F8FAFC). No text on the image. 1200x630px for OG compatibility. No stock photo feel.
4. Save the image locally, then upload to Sanity as the post's featured image
5. Use the same image as the OG image for social sharing

**API call example (Python):**
```python
import google.generativeai as genai
genai.configure(api_key=os.environ["GEMINI_API_KEY"])
model = genai.GenerativeModel("gemini-2.0-flash-exp")
response = model.generate_content(["Generate a blog header image: [description]"])
# Save response.parts[0].inline_data to file
```

**Fallback:** If Gemini output isn't right, use the Sanity MCP `generate_image` tool with a similar prompt.
